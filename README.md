

## Основные возможности

- Связывание элементов управления с свойствами модели (`ViewModel`)
- Двусторонняя и односторонняя привязка
- Преобразование типа данных с помощью конвертера
- Создание привязки динамически по событию (нажатию кнопки)

---

## Структура проекта

- **MainWindow.xaml** — описание интерфейса (разметка UI)
- **MainWindow.xaml.cs** — логика программы и настройка привязок
- **ViewModel** — класс модели с DependencyProperty для хранения данных
- **ColorToBrushConverter** — конвертер для преобразования `Color` в `Brush`

---

## Как использовать программу

1. **Работа с интерфейсом:**
   - Вводите текст в поле "TextBoxString". Связь двусторонняя, изменения отображаются в модели.
   - Отмечайте или снимайте галочку "CheckBoxBool". Связь двусторонняя.
   - Вводите число в "TextBoxInt" и "TextBoxDouble". Связь односторонняя (от модели к UI).
   - Цвет квадрата отображается согласно значению свойства `Color`. Можно изменить значение свойства в коде или через интерфейс (при добавлении).
   - Нажмите кнопку "Создать динамическую привязку" — в окне появится `TextBlock`, привязанный к свойству `String` модели. Это демонстрация динамической привязки.

---

## Как работает код

### Основные компоненты

- **ViewModel:** хранит свойства, связанные с UI, реализованные через `DependencyProperty`.
- **Привязки:** настроены в конструкторе `MainWindow`, связывая свойства модели с элементами управления.
- **Конвертер:** `ColorToBrushConverter` преобразует `Color` в `SolidColorBrush` для установки цвета `Rectangle`.
- **Динамическая привязка:** создается при нажатии кнопки и добавляется в визуальное дерево.
